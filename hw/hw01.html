Problem 1: Temperature Conversion Program (10 pts + 5 bonus)#

Write a Python script that converts temperature between scales given a user-specified numerical value and units (K, C, F).

Basic Version (10 pts):#

The program should:

  1. The user specifies a temperature value (e.g., 100.0) and its units (e.g., C for Celsius)

  2. Convert the input temperature to the two other temperature scales (e.g., C -> K and F)

  3. Print the results with clear descriptive text and units

Example:

  • Input: 100.0 and C

  • Output:

    100.0 C is equal to:
373.15 K
212.0 F

Bonus Version (+5 pts):#

Create a more robust version that can understand the temperature scale by the first letter only (case insensitive).

Your program should correctly interpret all of these as Kelvin:

  • K, k, Kelvin, KELVIN

Test your bonus version with these examples:

  • 300 with units K → should convert to C and F

  • 300 with units Kelvin → should convert to C and F

  • 300 with units k → should convert to C and F

Task 1: Functional Group Identification (5 pts)#

Write a Python program that takes a molecular formula as input (e.g., C5H12, C4H8) and identifies whether it corresponds to:

  • Alkane (CₙH₂ₙ₊₂)

  • Alkene (CₙH₂ₙ)

  • Alkyne (CₙH₂ₙ₋₂)

Requirements#

  1. Prompt the user to input a molecular formula as a string.

  2. Extract and print the number of carbon atoms (C) and hydrogen atoms (H).

  3. Use these values to identify the type of hydrocarbon.

Examples#

  • C2H4 → Alkene

  • C12H26 → Alkane

Task 2: Calculating the Molecular Weight of a Compound (No Loops or Conditionals) (10 pts)#

Write a Python program that calculates the molecular weight of a compound containing only carbon (C), hydrogen (H), and oxygen (O), given its stoichiometric formula (e.g., C2H6O, CH4, C6H12O6).

Use the following atomic weights:

  • Carbon (C) = 12 g/mol

  • Hydrogen (H) = 1 g/mol

  • Oxygen (O) = 16 g/mol

Requirements#
